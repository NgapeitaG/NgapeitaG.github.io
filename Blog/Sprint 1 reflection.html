Track Changes - Reflection

How would you describe stage and commit to your non-tech-savvy friend?

Staging is when you submit a file before you're about to change it and save it as a new version. It's the step you do to a file once everything has been changed, you move it to the staging area and then the changes are ready to be saved and recorded for your file.

Committing is where you submit a file so that you can record the changes that have been made to that file. Once you commit your file, it will have a special set of numbers and letters called a SHA which is a fancy ID to categorise your changes you made to your file. When you commit a file, you're putting on record the actual changes you made to your file you're committing to the git gods.

Setting Remotes and Syncing Forks - Reflection

What is a remote?

A remote is a place where you can save versions of your work. It enables you to save the work locally on your computer from a server, edit it and push your edited version from your computer back to the server,with the original project.

How are they set up?

Firstly, you have to check and see what remotes you already have. To do this type git remote -v. This will show all your remotes. Now that you can see all your remotes, you can decide whether to delete any. If you're satisfied with your remotes and don't need to add any, this step is not necessary. To add a remote type git remote add Then type git remote -v and then you will see that your new remote has been added. How would you explain a remote to a non-technical person using an analogy? A remote is a place where you have your files stored on a server. It's a remote place where you can access files.

What did you learn about flipped classroom?

A flipped classroom is not the tradition classroom where you sit in class and listen to a teacher speaking for long periods of time. It is usually online based learning where the student can watch tutorials and has access to resources where they can stop and start and go back to learning when they feel like it. The student has the ability to learn at their own pace and go back to any information they don't understand, until they get the concept. Flipped classroom is an outside of the class learning experience where you learn things before you initially work with a teacher. It's a prep/primer class usually web-based before you go into the physical classroom and learn from your teacher.
